# Controls for Pathfinder
## Generic controls:
    - G: Clear board
    - B: Run algorithm
    - Escape: Settings
## In Mouse mode:
    - Left click: Walls
    - Ctrl + Left click: Obstacles
    - Middle click: Starting point
    - Ctrl + Middle click: Destination
    - Right click: Delete Tiles
## In Trackpad mode:
    - Click: Starting point
    - Ctrl + Click: Destination
    - W + Click: Wall
    - 1 + Click: Obstacles
    - Double tap: Delete Tiles
